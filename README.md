# BuildForge-release
🇬🇧 English Description
BuildForge – A Powerful Build Automation Tool
BuildForge is a cross-platform desktop application that simplifies the process of building Python projects. With a modern and intuitive user interface, it allows developers to quickly convert their source code into standalone executable files.

✨ Features:
One-Click Build – Select your project folder and build with a single click.

Multi-Language UI – Supports 7 languages (Persian, English, Chinese, Russian, Arabic, Japanese, Korean).

Build Modes – Choose between Debug, Release, and Optimized builds.

Customizable Output – Set output file name, version, and icon.

File Selection – Select specific files from your project to include in the build.

Progress Bar with ETA – Real-time progress indicator with estimated time remaining.

Secure Code – Built-in obfuscation and bytecode compilation for code protection.

Cross-Platform – Works on Windows (installer available), with support for Linux and macOS coming soon.

🛠️ How to Use:
Select your project folder (containing main.py, app.py, or run.py).

Choose build type (Debug, Release, Optimized).

Set output name, version, and optional icon.

Click Start Build.

Find your executable in the dist folder.

📦 Requirements:
Python 3.x installed on your system.

PyInstaller (automatically installed if missing).

🇮🇷 توضیحات فارسی
BuildForge – ابزار ساخت خودکار پروژه‌های برنامه‌نویسی
BuildForge یک اپلیکیشن دسکتاپ چندسکویی است که فرآیند ساخت پروژه‌های پایتون را ساده می‌کند. با رابط کاربری مدرن و ساده، به توسعه‌دهندگان اجازه می‌دهد تا سریعاً کدهای خود را به فایل‌های اجرایی مستقل تبدیل کنند.

✨ قابلیت‌ها:
ساخت با یک کلیک – پوشه پروژه را انتخاب کنید و با یک کلیک بیلد را انجام دهید.

رابط کاربری چندزبانه – پشتیبانی از ۷ زبان (فارسی، انگلیسی، چینی، روسی، عربی، ژاپنی، کره‌ای).

حالت‌های ساخت – انتخاب بین حالت‌های Debug، Release و Optimized.

خروجی قابل تنظیم – تنظیم نام فایل خروجی، نسخه و آیکون.

انتخاب فایل‌ها – انتخاب فایل‌های خاص از پروژه برای قرار گرفتن در بیلد.

نوار پیشرفت با زمان تخمینی – نمایش پیشرفت واقعی با زمان باقی‌مانده.

کد امن – ابهام‌سازی داخلی و تبدیل به بایت‌کد برای محافظت از کد.

چندسکویی – کار بر روی ویندوز (نصب‌کننده موجود)، پشتیبانی از لینوکس و مک در آینده.

🛠️ نحوه استفاده:
پوشه پروژه خود را انتخاب کنید (شامل فایل main.py، app.py یا run.py).

نوع بیلد را انتخاب کنید (Debug، Release، Optimized).

نام خروجی، نسخه و آیکون دلخواه را تنظیم کنید.

روی شروع بیلد کلیک کنید.

فایل اجرایی خود را در پوشه dist پیدا کنید.

📦 نیازمندی‌ها:
Python 3.x نصب شده روی سیستم.

PyInstaller (در صورت نبود، به‌طور خودکار نصب می‌شود).
